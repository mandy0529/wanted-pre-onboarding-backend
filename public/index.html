API 문서

User

  • 1. 회원가입 
    POST /api/v1/user/register 
         -d '{ "email": "test@gmail.com", "password": "password1234" }
  • 2. 로그인 
    POST /api/v1/user/login 
         -d '{ "email": "test@gmail.com", "password": "password1234" }

Post

  • 1. 게시글 글쓰기 
    POST /api/v1/post 
        -H "Authorization: Bearer ${token}"" 
        -d '{ "title": "새로운 게시글", "content": "이것은 새로운 게시글 내용입니다." }

    * 게시글을 작성할 때는 Bearer Token이 필요합니다. Authorization 헤더에 유효한 Bearer Token을 포함하여 요청하세요.

  • 2. 모든 게시글 불러오기 (pagination 포함) 
    GET /api/v1/post?page=1
  • 3. 해당 게시글 불러오기 
    GET /api/v1/post/:id
  • 4. 해당 게시글 수정하기 
    PATCH /api/v1/post/:id 
          -d '{ "title": "edit title", "content": "edit content" }

    * 게시글을 수정할 때는 Bearer Token이 필요합니다. Authorization 헤더에 유효한 Bearer Token을 포함하여 요청하세요.

  • 5. 해당 게시글 삭제하기 
    DELETE /api/v1/post/:id

    * 게시글을 삭제할 때는 Bearer Token이 필요합니다. Authorization 헤더에 유효한 Bearer Token을 포함하여 요청하세요.
